## 📦 Installation

1. Cloner le dépôt :
```bash
git clone https://github.com/arnoldnzama/nsp.git
```

2. Configurer le fichier `config.php` avec vos paramètres email

3. Uploader les fichiers sur votre serveur web

4. Assurez-vous que PHP est installé et que la fonction `mail()` est activée

## 📧 Configuration Email

Le formulaire de contact envoie les demandes à : **admin@nsp.cd**

Pour modifier l'adresse de réception, éditez le fichier `config.php` :
```php
define('EMAIL_TO', 'votre-email@domaine.com');
```
